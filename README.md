# GPS-1-💰 Google Pay Expense Sharing System
Developed a Python-based expense sharing application that simplifies group expense management. The system allows users to add participants, record expenses, and automatically calculate individual shares and balances. Implemented logic using Python data structures like lists and dictionaries to ensure accurate and efficient bill splitting. This project demonstrates problem-solving skills, financial calculation logic, and practical application development.
Technologies Used: Python, OOP Concepts
